# Applaunch

Applaunch is an opinionated application launcher and dock for the Niri compositor.
It was inspired by GNOME desktop workflow and was designed to work with convertible laptops and tablet PCs.

# Dependencies
- glib-2.0
- gobject-2.0
- gtk4
- libadwaita-1
- gtk4-layer-shell-0
- gio-unix-2.0
- gio-2.0
- appstream
- json-glib-1.0
- gee-0.8
